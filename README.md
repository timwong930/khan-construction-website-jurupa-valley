# Khan Construction Co. website starter

## Public business snapshot
- **Company:** Khan Construction Co.
- **Location:** Jurupa Valley, CA
- **Category:** Construction
- **Website:** https://khanconstruction.com/
- **Social:** none found in the public scan

## What they do
A construction and roofing company serving the Inland Empire with services that span new builds, remodeling, and roofs.

## Why this looks like a good opportunity
- The homepage title is bare-bones and the meta description is missing, which weakens search visibility.
- The company offers several high-value services, but the site does not yet package them into clear local landing pages.
- A stronger structure could separate roofing, ADUs, and remodeling into focused pages with project examples and quote CTAs.

## Suggested outreach / redesign angle
Lead with an SEO-first restructuring: clearer service pages, stronger meta data, and a better local portfolio presentation.

## Starter site included in this repo
This repo includes a simple, responsive starter site in `index.html` and `styles.css` built only from public information.

### Draft structure
- **ADUs and additions** — Planning and build support for property owners who need more space.
- **Roofing** — Flat, shingle, tile, and metal roof work with clear estimate paths.
- **Remodeling** — Interior upgrades and broader renovation projects.
- **Projects** — A concise gallery that shows finished work by service type.
